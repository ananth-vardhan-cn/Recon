# Recon Frontend

## Setup
1. Navigate to this directory: `cd frontend`
2. Install dependencies: `npm install` (Already done)

## Running the App
1. **Start the Backend:**
   From the root `geovista_clone` directory:
   ```bash
   uvicorn server:app --reload --host 0.0.0.0 --port 8000
   ```

2. **Start the Frontend:**
   From the `frontend` directory:
   ```bash
   npm run dev
   ```

3. Open your browser to the Local URL shown (usually `http://localhost:5173`).
